الاخراج

اسم الدالة الصيغة العامة الكود النتيجة الشرح
print print(value, ...) print("Hello, World!") Hello, World! تطبع النص أو القيم إلى وحدة الإخراج القياسية.
format str.format(args, ...) print("Hello, {}".format("World")) Hello, World تنسيق النص باستخدام تعبيرات.
f-string f"{expression}" name = "World"; print(f"Hello, {name}") Hello, World تنسيق النص باستخدام تعبيرات في بايثون 3.6 وما بعده.
file write file.write(string) with open('file.txt', 'w') as file: file.write("Hello, World!") (لا شيء) كتابة نص إلى ملف.
file writelines file.writelines(list) with open('file.txt', 'w') as file: file.writelines(["Hello\n", "World\n"]) (لا شيء) كتابة قائمة من الأسطر إلى ملف.
csv writer csv.writer(file) import csv; with open('file.csv', 'w', newline='') as file: writer = csv.writer(file); writer.writerow(['Hello', 'World']) (لا شيء) كتابة صف إلى ملف CSV.
json dump json.dump(obj, file) import json; data = {"hello": "world"}; with open('file.json', 'w') as file: json.dump(data, file) (لا شيء) كتابة كائن بايثون إلى ملف JSON.
logging logging.info(msg) import logging; logging.basicConfig(level=logging.INFO); logging.info("Hello, World!") INFO:root:Hello, World! تسجيل رسالة معلومات إلى وحدة الإخراج القياسية أو ملف.
pprint pprint.pprint(object) import pprint; data = {'hello': 'world'}; pprint.pprint(data) {'hello': 'world'} طباعة كائنات بايثون بشكل منسق وسهل القراءة.
sys.stdout.write sys.stdout.write(string) import sys; sys.stdout.write("Hello, World!") Hello, World! كتابة نص إلى وحدة الإخراج القياسية بدون إضافة سطر جديد.
